# AN Boutique Website

A modern and responsive boutique website designed for showcasing fashion collections, designer sarees, women's fashion, and accessories.

## Features

- Responsive Bootstrap 5 design
- Modern Hero Section
- About Us section
- Product Collections with Modal Popups
- Gallery Section
- Contact Form
- WhatsApp Floating Button
- Mobile-Friendly Layout
- Smooth Scrolling Navigation
- Attractive UI with Fashion Theme

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5.3.3
- Font Awesome 6.5.2
- JavaScript

## Installation

1. Download or clone the project.

```bash
git clone https://github.com/yourusername/an-boutique.git
```

2. Navigate to project directory.

```bash
cd an-boutique
```

3. Open `index.html` in your browser.

## Customization

### Update Boutique Name

Search:

```html
AN Boutique
```

Replace with your boutique name.

### Update Contact Details

Edit the Contact section:

```html
+91 9876543210
an_boutique@gmail.com
Vijayawada, Andhra Pradesh
```

### Update WhatsApp Number

Change:

```html
https://wa.me/919876543210
```

To your WhatsApp number.

### Update Images

Replace the image files:

- logo.jpeg
- designer sarees.png
- women's fashion.png
- accessories pic.png
- saree types.png
- dresses.png
- accessories.png

with your own boutique images.
