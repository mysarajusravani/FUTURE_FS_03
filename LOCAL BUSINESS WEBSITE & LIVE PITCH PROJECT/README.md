# Elegant Boutique Website

## Overview

Elegant Boutique is a responsive fashion boutique website developed using HTML, CSS, JavaScript, Bootstrap, and Font Awesome.

The website showcases boutique collections, gallery images, store details, and a contact form with a modern and elegant user interface. 

---

## Features

- Fully Responsive Design
- Attractive Hero Section
- About Us Section
- Fashion Collections Showcase
- Image Gallery
- Festive Sale Offer Section
- Collection Detail Modals
- Contact Form with Validation
- WhatsApp Quick Contact Button
- Bootstrap Components
- Modern UI Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5.3.3
- Font Awesome 6.5.2

## How To Run

### Method 1

1. Download the project.
2. Extract the folder.
3. Open `index.html` in your browser.

### Method 2 (Recommended)

Using VS Code:

1. Install Live Server Extension.
2. Open project folder.
3. Right Click on `index.html`.
4. Click **Open with Live Server**.
